*您需要熟悉一些linux指令
*苹果M1芯片暂不支持

1. 本地单点部署

1.安装Docker
  • 官方安装文档
2.安装MySQL
  • 官方安装文档
  • 安装成功后请创建auth_prod数据库并导入数据, 示例 mysql -u root -p auth_prod < init.sql
3.配置Redis与RabbitMQ容器
  • 切换至目录: cd mt-auth/script/deploy
  • 启动容器: sh infra.sh
4.配置mt-auth启动脚本
  • 启动mt-access, 脚本请移步这里, *启动mt-access后请等待150秒再启动mt-proxy
  • 启动mt-proxy, 脚本请移步这里
5.最后(可选)
  • 脚本配置完成后，您可以直接移步这里
6.启动前端
  • cd ./mt-ui
  • npm install && npm start
  • 前端成功安装并启动后，访问http://localhost:4300
